# Fitness_Tracker_HW

This is the Fitness Tracker assignment. User will be able to logg in their workout routies and keep track of all their fitness goals. the goals are tracked by days of the week, as well as the duration of each excerise.

Users can select the type of excerise, how long they want to excerise and keep track of all of their stats.

This app allows to allows uses to stop and continue their workouts as well.
